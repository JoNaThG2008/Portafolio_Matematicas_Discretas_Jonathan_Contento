# 📖 2.1 Operaciones Booleanas

El Álgebra Booleana es una rama de la Matemática Discreta que permite trabajar con variables lógicas que únicamente pueden tomar dos valores posibles: **0 (falso)** y **1 (verdadero)**.

Estas operaciones son fundamentales en el área de computación, ya que permiten representar condiciones lógicas, diseñar circuitos digitales y optimizar expresiones utilizadas en sistemas electrónicos y programas informáticos.

Las principales operaciones booleanas son:

| Operación | Símbolo | Descripción |
|:---------|:-------:|-------------|
| AND | ∧ o · | El resultado es verdadero cuando todas las entradas son verdaderas. |
| OR | ∨ o + | El resultado es verdadero cuando al menos una entrada es verdadera. |
| NOT | ¬ o ' | Invierte el valor lógico de una variable. |

---

### ✏️ Ejemplo

Sean las variables:

- **A = 1**
- **B = 0**

Entonces:

| Expresión | Resultado | Interpretación |
|:---------|:---------:|----------------|
| **A · B** | 0 | La operación AND requiere que ambas variables sean 1. |
| **A + B** | 1 | La operación OR es verdadera cuando una variable es 1. |
| **A'** | 0 | La operación NOT invierte el valor de A. |

Estas operaciones permiten construir expresiones lógicas utilizadas en el análisis y diseño de sistemas digitales.

---

# 📖 2.2 Propiedades adicionales del Álgebra Booleana

El Álgebra Booleana posee diferentes propiedades que permiten transformar y simplificar expresiones lógicas. Estas leyes ayudan a reducir operaciones y obtener circuitos digitales más eficientes.

Entre las principales propiedades se encuentran:

| Propiedad | Expresión |
|:----------|:----------|
| Identidad | A + 0 = A  /  A · 1 = A |
| Dominación | A + 1 = 1  /  A · 0 = 0 |
| Idempotencia | A + A = A  /  A · A = A |
| Complemento | A + A' = 1  /  A · A' = 0 |
| Doble negación | (A')' = A |
| Absorción | A + A·B = A |

---

### ✏️ Ejemplo de simplificación

Expresión inicial:

**A + A · B**

Aplicando la propiedad de absorción:

**A + A · B = A**

La aplicación de estas propiedades permite obtener expresiones equivalentes más sencillas, facilitando el diseño y optimización de circuitos lógicos.
