# 📖 3.1 Teoría de Grafos

La teoría de grafos es una rama de la Matemática Discreta que estudia estructuras formadas por un conjunto de **vértices o nodos** y un conjunto de **aristas** que representan las conexiones entre ellos.

Los grafos permiten representar diferentes situaciones de la vida real y problemas computacionales, como redes sociales, sistemas de transporte, conexiones entre dispositivos y relaciones entre diferentes elementos.

Un grafo está compuesto por:

| Elemento | Descripción |
|:---------|-------------|
| **Vértices (V)** | Representan los elementos o puntos principales del grafo. |
| **Aristas (E)** | Representan las conexiones existentes entre los vértices. |

---

### ✏️ Ejemplo

Sea el conjunto de vértices:

**V = {A, B, C, D}**

Y las conexiones:

**E = {(A,B), (B,C), (C,D)}**

Esta representación muestra cómo los diferentes vértices se encuentran relacionados mediante las aristas.

---

# 📖 3.2 Algoritmos para Grafos

Los algoritmos para grafos permiten analizar, recorrer y resolver problemas relacionados con estas estructuras. Son utilizados para encontrar caminos, determinar conexiones y explorar la información almacenada dentro de un grafo.

Algunos de los algoritmos más utilizados son:

| Algoritmo | Descripción |
|:----------|-------------|
| **Búsqueda en profundidad (DFS)** | Explora un camino hasta llegar al final antes de regresar y analizar otras conexiones. |
| **Búsqueda en amplitud (BFS)** | Recorre los nodos por niveles, visitando primero los elementos más cercanos. |
| **Algoritmo de Dijkstra** | Permite encontrar el camino más corto entre dos vértices. |

---

### ✏️ Ejemplo

En una red de computadoras, un algoritmo de grafos puede utilizarse para encontrar la ruta más eficiente entre dos dispositivos conectados mediante diferentes enlaces.

Estos algoritmos son importantes para optimizar procesos y resolver problemas relacionados con redes y conexiones.

---

# 📖 3.3 Árboles

Los árboles son estructuras especiales de grafos que no contienen ciclos y poseen una organización jerárquica. Están formados por nodos conectados mediante ramas, donde existe un nodo principal denominado **raíz**.

Los árboles tienen diversas aplicaciones en computación, como la representación de sistemas de archivos, bases de datos, árboles de búsqueda y algoritmos de decisión.

## Características principales:

- Poseen un nodo raíz.
- Cada nodo puede tener cero o más nodos hijos.
- No contienen ciclos.
- Existe un único camino entre dos nodos.

---

### ✏️ Ejemplo

```
          A
        /   \
       B     C
      / \
     D   E
```

En este árbol:

- **A** representa la raíz.
- **B y C** son nodos hijos.
- **D y E** pertenecen a un nivel inferior del árbol.
